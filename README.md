# Sha Posh Clothing Store Website

A responsive, static website for Sha Posh clothing store in Islamabad, Pakistan. This website is built using pure HTML, CSS, and JavaScript with no frameworks or backend dependencies.

## Features

- Fully responsive design (mobile-first approach)
- Clean, modern fashion store UI
- SEO-friendly semantic HTML structure
- Interactive elements with vanilla JavaScript
- GitHub Pages compatible with relative paths
- No external dependencies (except Font Awesome icons)

## Website Sections

1. **Hero Section** - Store name, tagline, rating, and call-to-action buttons
2. **About Section** - Introduction to Sha Posh with key features
3. **Reviews Section** - Customer testimonials with star ratings
4. **Contact & Location** - Store address, hours, phone, and map placeholder
5. **Social Media** - Links to social platforms
6. **Footer** - Complete business information and copyright
